## git tutorial

- how to clone a repo?
    - Just  copy the url from the http
    - then go to `Terminal` then type `git clone (paste the url)`

- the three important comands are
    - `git add .`
    - `git commit -m "message" `
    - `git push`

- How to open `terminalin VS Code`
    - `ctrl J`

- How to hard reload?
    -`Ctrl+ shift+r`
